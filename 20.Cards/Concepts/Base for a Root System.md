## Definition

> [!definition] Definition: 
> A subset $B$ of $R$ is a **base** for the root system $R$ if
> 
> (B1) $B$ is a vector space basis for $E$, and
> 
> (B2) every $\beta \in R$ can be written as $\beta = \sum_{\alpha \in B} k_{\alpha} \alpha$ with $k_{\alpha} \in \mathbf{Z}$, where all the non-zero coefficients $k_{\alpha}$ have the same sign.
> 
> We say that a root $\beta \in R$ is **positive with respect to $B$** if the coefficients given in (B2) are positive, and otherwise it is **negative with respect to $B$**.

> [!remark] Remark: 
> Note that from [[pps - Root's Additivity(Subtractibility) when strictly Obtuse(Acute)|Root's Additivity(Subtractibility) when strictly Obtuse(Acute)]] if $B$ is a base for a root system, then the angle between any two distinct elements of $B$ is obtuse (that is, at least $\pi/2$).

## Exsistence

> [!theorem] Theorem: 
> Every root system has a base.

> [!remark] Remark: 
> * 事实上, 取一个不包含任何根的超平面(或者不落在任何其它根定义的超平面的一个向量 $z$),规定正侧和负侧, 令 $R^{+ }$为 $R$中落在超平面正侧的根. 则 $R^{+ }$中不能两个和的根全体就是一个基.
> * It is seen from the proof that the base is given by a [[dfn - Regular with respect to a Root System|regular]] element $z\in E$, it may be denoted by $\Delta \left(z\right)$

> [!proofsketch] Sketch of Proof: 
> * 令 
>  $$ B= \left\{  r \in R^{+ }: r\text{ is not the sum of any two elements in } R^{+ }\right\} $$
> * 说明(B2)只需要 $R^{+ }$中元素都能写成非负整B-线性组合, 进一步地只需要考察 $R^{+ }\setminus B$.
> * 为此, 用**能拆成两个$R^{+ }$-和**的性质递归地传递**不能写成非负整B-线性组合**的性质, 得到一列根 $\left\{ \beta _{i} \right\}$, 使得 $\left(z,\beta _{i}\right)$严格递减, 违背 $R^{+ }$的有限性.
> * 为了说明线性无关性, 考察线性组合$\sum _{\alpha \in B}r_{\alpha }\alpha = 0$. 
> * **玄学地讲**, $B$中的根是两两背离但是都靠近$z$的. 正负系数的背离会响应根的背离, 导致能量产生, 因此系数是不能存在的.
> * 收集正系数部分 $x= \sum _{\alpha :r_{\alpha }> 0}r_{\alpha }\alpha$, 剩下负系数部分 $-x= \sum _{\beta :r_{\beta }< x}\left(-r_{\beta }\right)\beta$. 
> * 基中两两根成钝角$\left(\alpha ,\beta \right)\le 0$([[pps - Root's Additivity(Subtractibility) when strictly Obtuse(Acute)#Corollary|Root's Additivity(Subtractibility) when strictly Obtuse(Acute)]]), 导致 $\left(x,x\right)\le 0$. 得到正负系数部分本身分别为零($x= 0$).
>  $(\cdot ,z)$会赋予 $R^{+ }$中元素正的加权, 强迫系数为零.


## Properties

> [!theorem] Theorem: 
> Let $\Delta$ be a base of $\Phi$
> 1. If $\gamma \in E$ is regular, there exists $\sigma \in W$ such that $\left(\sigma \left(y\right),\alpha \right)> 0$ fo all $\alpha \in \Delta$(so W acts transitively o Weyl chambers).
> 2. If $\Delta ^{\prime}$ is another base of $\Phi$, then $\sigma \left(\Delta ^{\prime} \right)= \Delta$ for some $\sigma \in W$.
> 3. If $\alpha$ is any root, thre exists $\sigma \in W$ such that $\sigma \left(\alpha \right)\in \Delta$
> 4. $W$ is generated by the $\sigma _{\alpha }\left(\alpha \in \Delta \right)$.
> 5. If $\sigma \left(\Delta \right)= \Delta$, $\sigma \in W$, then $\sigma = 1$.(so $W$ acts simply transitively on bases).

## Exmaple

> [!example] Example: 
> Let $R=\{\pm(\varepsilon_i-\varepsilon_j):1\le i<j\le\ell+1\}$ be a root system in $E= \operatorname{span}R$. Let $\alpha_i=\varepsilon_i-\varepsilon_{i+1}$ for $1\le i\le\ell$. Then $B=\{\alpha_1,\dots,\alpha_\ell\}$ is a base for $R$ and find the positive roots.

> [!proof] Proof:
